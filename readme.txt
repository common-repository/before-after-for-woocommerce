== Installation ==

1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. OR, Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'eBEAF'.
4. Install and activate the plugin.
5. You can also check this step by step <a href="https://youtu.be/FHvrLiMb5Wg">Youtube Video</a>.

= What is Twenty Twenty Script? =

TwentyTwenty is a script developed by Zurb team. If you want to highlight the differences between two images then this script is the best to use. TwentyTwenty, a visual diff tool, makes it easy to spot them! As per their description, it works by combining two images on top of each other.

= What is Before After Image? =

Image Before & After means comparing multiple images before and after situation. You can showcase how an image was before and how it is now. For example, in case of dental patient, a doctor can showcase how the patientâ€™s teeth was before and how it is now after the treatment. More procisely, showing dental before after image.

= What is eBEAF? =

eBEAF is a WooCommerce based plugin to create WordPress Before After Slider in WooCommerce. Want to show comparison of two images? This plugin helps you to easily create before and after image sliders.
